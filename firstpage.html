Pterocarpus Angolensis
This tree is a very unique type of tree this is because if the log of the tree is cut it will bleed red just like us humans. The main reson for its red bleeding is because of, Tannis which is type of chemical which makes the sap of the tree red this type of chemical is often found in wines.
